## Getting Started

### Download and Install

Download the Identity Co mobile app from your device's app store:

#### Unable to Register

**Problem**: Registration code not working

**Solution:**
1. Verify you entered the code correctly
2. Check if the code has expired
3. Contact your administrator for a new code
4. Ensure you have a stable internet connection

#### Biometric Authentication Failing

**Problem**: Face ID or fingerprint not recognized

<div className="screenshot-group">
  <img src="/img/user-guide/21-reset-biometrics.png" alt="Reset biometrics screen" />
  <p className="caption">Re-register your biometric data</p>
</div>

**Solution:**
1. Clean your camera lens or fingerprint sensor
2. Ensure adequate lighting (for Face ID)
3. Re-register your biometric data:
   - Go to **Settings** > **Security**
   - Tap **Update Biometrics**
   - Follow the setup process

#### Not Receiving Push Notifications

**Problem**: No notifications appearing

**Solution:**
1. Check device notification settings
2. Ensure Identity Co has notification permissions
3. Verify internet connection
4. Sign out and sign back in
5. Reinstall the app if issue persists

#### App Crashing or Freezing

**Problem**: App won't open or freezes during use

**Solution:**
1. Force close the app
2. Restart your device
3. Update to the latest app version
4. Clear app cache (Settings > Storage)
5. Reinstall the app
